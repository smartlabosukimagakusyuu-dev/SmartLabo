# PROMPTS

**AI用プロンプト集**

---

## このフォルダの目的

ChatGPT・Claude Code・Codex にタスクを依頼する際に使う、再利用可能なプロンプトのテンプレートを保管します。プロンプトを個人の頭の中やチャット履歴に留めず、会社の資産として蓄積することを目的とします。

AI利用の共通ルールは [PROJECT_BIBLE/30_AI_Rules.md](../PROJECT_BIBLE/30_AI_Rules.md) を必ず参照してください。

---

## サブフォルダ構成

| フォルダ | 内容 |
|---|---|
| [CLAUDE/](CLAUDE/README.md) | Claude Code / Claude 向けのプロンプト・設定(CLAUDE.md のひな形など) |
| [CODEX/](CODEX/README.md) | Codex 向けのプロンプト・自動化タスク定義 |
| [CHATGPT/](CHATGPT/README.md) | ChatGPT 向けの企画・文章作成プロンプト |

---

## 利用ルール

- 良い結果が得られたプロンプトは、個人のメモに留めずこのフォルダに保存してください。
- プロンプトには「用途」「使い方」「注意点」を簡潔に添えてください。
- 顧客の機密情報を含む具体例をプロンプトに含めないでください(テンプレートは汎用化する)。

---

## Version管理

- プロンプトを改善した場合は、上書きせず変更履歴がわかる形でコミットしてください(Gitの差分で十分追跡可能です)。

---

## 更新ルール

- [PROJECT_BIBLE/](../PROJECT_BIBLE/README.md) の内容(Mission/Vision/Value/Rules)が変更された場合、それを前提とするプロンプトも見直してください。
- 陳腐化した・使われなくなったプロンプトは [ARCHIVE/](../ARCHIVE/README.md) に移動してください。

---

## Git運用ルール

- 通常のドキュメント同様、直接コミットで構いません。大きなテンプレート改修時のみレビューを推奨します。
