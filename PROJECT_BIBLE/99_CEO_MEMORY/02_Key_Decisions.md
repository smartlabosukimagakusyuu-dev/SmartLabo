# 02. Key Decisions — 重要な経営判断の記録

## このドキュメントについて

会社にとって重要だった意思決定について、「何を選んだか」だけでなく「なぜそれを選んだか」「何を選ばなかったか」を残します。後から振り返ったときに、当時の判断の背景を思い出せるようにするための記録です。

> *記入ガイド: 創業の形態、最初のプロダクト領域の選定、資金調達の方針、人を採用する/しない判断、大きな方針転換など、「なぜそう判断したか」を書き残す価値がある意思決定があれば、この表に追記してください。*
