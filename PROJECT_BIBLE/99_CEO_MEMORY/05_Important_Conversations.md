# 05. Important Conversations — 重要な会話の記録

## このドキュメントについて

顧客、投資家、パートナー、社員などとの会話の中で、会社の方向性に影響を与えた・示唆に富んでいた・記録しておく価値があると感じた会話を記録します。[MEETING/](../../MEETING/README.md) が業務としての議事録であるのに対し、ここは代表個人の視点で「印象に残った・重要だと感じた」会話を残す場所です。

> *記入ガイド: いつ、誰との、どんな会話だったか。その会話から何を得たか・何が変わったかを記入してください。相手の発言をそのまま引用しても構いません。*
