# 03. Value — 大切にする価値観

会社としての意思決定、プロダクトの設計、AIへの指示、すべての場面でこの価値観を判断基準とします。迷ったときは、この5つに立ち返ってください。

---

## 1. シンプルさ (Simplicity)

複雑な機能より、シンプルで確実に機能するものを選びます。「一つ足す前に、一つ削れないか」を常に問います。

## 2. 誠実さ (Integrity)

顧客・取引先・仲間に対して、誇張のない誠実な情報提供を行います。AIが生成する内容も含め、事実に基づかない主張はしません。

## 3. スピード (Speed)

完璧な計画より、小さく早く動いて検証することを優先します。ただし、スピードは品質を犠牲にする言い訳にはしません。

## 4. 当事者意識 (Ownership)

「誰かがやるだろう」ではなく、気づいた人が責任を持って動きます。AIエージェントに作業を依頼する場合も、依頼者が結果に責任を持ちます。

## 5. 余白を大切にする (Space to Think)

情報もデザインも、詰め込みすぎないことを大切にします。余白があることで、本当に重要なものが際立ちます。この考え方は [07_UI_UX_Rules.md](07_UI_UX_Rules.md) のデザイン原則にも直結しています。

---

## 価値観の使い方

- 会議やレビューで意見が対立したとき、この5つのどれを優先するかで議論する
- 新機能の要否を判断するとき、「1. シンプルさ」に反していないか確認する
- AIに生成させたコンテンツを公開する前に、「2. 誠実さ」に照らして事実確認する
