# DESIGN_ASSETS / Colors

**カラースウォッチ・プレビュー素材**

---

## このフォルダの目的

カラーパレットのスウォッチファイル(`.ase`、`.clr`等、デザインツール用のカラー定義)やプレビュー画像を格納します。**正式なカラーコードの定義そのものは** [BRAND/colors/palette.md](../../BRAND/colors/palette.md) **にあり、このフォルダはその実データ(ツール用ファイル)置き場です。**

---

## 利用ルール

- 格納する色は、必ず [BRAND/colors/palette.md](../../BRAND/colors/palette.md) の8色(Primary Navy / Smart Blue / Light Blue / Accent Blue / White / Light Gray / Gray / Dark Gray)と一致させてください。
- Smart Blue・Light Blue・Accent Blueは独立した色ではなく「Smart Blue Family」として一つのグループにまとめて管理してください。

---

## Version管理

- カラーパレットが変更された場合は、[BRAND/colors/palette.md](../../BRAND/colors/palette.md) を先に更新し、このフォルダのファイルを同期させてください。

---

## 更新ルール

- 色の変更はブランドの根幹に関わるため、[PROJECT_BIBLE/00_Foundation/07_Brand_Identity.md](../../PROJECT_BIBLE/00_Foundation/07_Brand_Identity.md) の更新ルール(経営層承認)に従ってください。

---

## Git運用ルール

- テキストベースのファイル(`.ase`等)はそのままコミットして構いません。
