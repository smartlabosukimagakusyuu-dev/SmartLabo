# DESIGN_ASSETS/01_HERO/Overlays

**Hero用オーバーレイ・グラデーション素材(補助アセット)**

---

## このフォルダの位置づけ

2026-07-03、CEOがAI_WORKSPACE経由で提供したHero背景素材シート([../Backgrounds/](../README.md) 参照)に同梱されていた、補助的なオーバーレイ・グラデーション素材です。単体の背景写真の上に重ねて使うことを想定した透過・半透過素材です。

## ファイル一覧

| ファイル | 種別 | 内容 |
|---|---|---|
| `overlay_network_light.webp` | オーバーレイ | 控えめなAIネットワーク(点+線)のライトなバリエーション |
| `overlay_digital_worldmap.webp` | オーバーレイ | ドット絵調のワールドマップ |
| `overlay_wave_line.webp` | オーバーレイ | 波状の光の筋 |
| `gradient_blue.webp` | グラデーション | ブルー系グラデーション |
| `gradient_darkblue.webp` | グラデーション | ダークブルー系グラデーション |
| `gradient_blue_white.webp` | グラデーション | ブルー×ホワイトのグラデーション |

## 利用ルール

- これらは単体では使用せず、[../Backgrounds/](../README.md) の背景写真、または既存のHero実装(CSS)と組み合わせて、テキストの可読性を補強する目的で使用してください。
- 現時点(2026-07-03)ではHomepage実装では未使用です。今後、他ページ(LP・資料等)で背景写真を使う際に活用を検討してください。
