# AI_WORKSPACE/ARCHIVE

**不採用・不要になった素材の保管(理由付き)**

---

## 役割

検討の結果、DESIGN_ASSETSへ登録しないと判断した素材を保管する場所です。単に削除せず、**「なぜ不採用だったか」を将来のAI・担当者が繰り返し確認しなくて済むように**、理由を記録して保管します。

例: 禁止ビジュアル(六角形背景等、[PROJECT_BIBLE/00_Foundation/07_Brand_Identity.md](../../PROJECT_BIBLE/00_Foundation/07_Brand_Identity.md) 参照)に該当する素材、方針転換により使われなくなった旧世界観の素材など。

---

## 運用ルール

- 素材を移動する際は、同名の `.md` メモに「不採用の理由」を必ず記録してください(記載例は [DESIGN_ASSETS/01_HERO/Backgrounds/hero_background_05_waterfront_city.md](../../DESIGN_ASSETS/01_HERO/Backgrounds/hero_background_05_waterfront_city.md) を参照)。
- ここにある素材は、実制作・実装で使用しないでください。
