# AI_WORKSPACE/INBOX/REFERENCE

上記のいずれにも分類しにくい参考資料(競合分析・ブランドキット参照画像・レイアウト参考等)を保存する場所です。

これらは多くの場合、直接DESIGN_ASSETSへ登録される「成果物」ではなく、Claude Codeが仕様検討の参考として読み込むための資料です。参考にした内容がPROJECT_BIBLEの仕様に反映された場合は、その旨をCHANGELOGに記録し、このフォルダの資料自体は [COMPLETED/](../../COMPLETED/README.md) または [ARCHIVE/](../../ARCHIVE/README.md) へ移動してください。詳細な運用ルールは [AI_WORKSPACE/README.md](../../README.md) を参照してください。
