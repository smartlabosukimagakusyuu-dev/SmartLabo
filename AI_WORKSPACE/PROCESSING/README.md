# AI_WORKSPACE/PROCESSING

**Claude Codeが確認・作業中の素材**

---

## 役割

[INBOX/](../INBOX/README.md) から、Claude Codeが実際に利用すると判断した素材を移動する場所です。ここにある素材は「作業中」であり、まだ [DESIGN_ASSETS/](../../DESIGN_ASSETS/README.md) への正式登録は完了していません。

---

## 運用ルール

- Claude Codeは、INBOXの素材を利用すると決めたら、対応するファイルをこのフォルダへ移動してから作業を開始してください(元のINBOXフォルダには残さない)。
- 作業が完了し、DESIGN_ASSETSへ正式登録したら、元データは [COMPLETED/](../COMPLETED/README.md) へ移動してください。
- 検討の結果、不採用と判断した場合は [ARCHIVE/](../ARCHIVE/README.md) へ移動し、理由を記録してください。
- このフォルダに長期間放置されたままの素材がないか、作業の節目で確認してください。
