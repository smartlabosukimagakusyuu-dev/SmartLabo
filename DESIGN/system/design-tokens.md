## UIコンポーネントの型

| コンポーネント | 仕様 |
|---|---|
| プライマリボタン | Navy地または Smart Blue地の塗りつぶし、白文字、角丸、ホバーアニメーション |
| セカンダリボタン | アウトライン(枠線のみ)、塗りつぶしなし |
| テキストリンク | Smart Blue文字、下線なし、矢印アイコンを添えてよい |
| 入力フィールド(通常) | 薄いグレーの枠線 |
| 入力フィールド(フォーカス) | Smart Blueの枠線でハイライト |
| カード | 白背景、タイトル・説明文・数値(金額・前月比等)を縦に配置 |
| ステータスバッジ | 完了(緑)/進行中(青)/未着手(グレー)/遅延(赤)の4種 |

## アイコンセット

線画(アウトライン)スタイルで統一する。現時点で定義されている代表アイコン:

- AI分析
- タスク管理
- 顧客管理
- 売上管理
- レポート
- 通知
- 設定
- セキュリティ

実際のSVGアイコンファイルは [DESIGN_ASSETS/Icons/](../../DESIGN_ASSETS/Icons/README.md) に格納します(現時点では仕様のみ。実ファイルは [PROJECT_BIBLE/50_TODO.md](../../PROJECT_BIBLE/50_TODO.md) の未対応タスクを参照)。

## ダッシュボードUIの構成要素

Smart Labo Works の管理画面(ダッシュボード)は、以下の要素で構成されます。詳細な設計指示は [PROMPTS/DESIGN/Dashboard.md](../../PROMPTS/DESIGN/Dashboard.md) を参照してください。

- サイドバーナビゲーション(ロゴ + メニュー: ダッシュボード / タスク管理 / 顧客管理 / 売上管理 / AI分析 / レポート / 設定)
- 上部の統計カード群(売上総額、稼働プロジェクト数、タスク完了率 等)
- データ可視化(売上推移の折れ線グラフ、タスク内訳のドーナツチャート)
- 「AIからの提案」パネル(AIによる気づき・提案の一覧)
- 「最近のアクティビティ」パネル(直近の操作履歴)
